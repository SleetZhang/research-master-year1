## 一、调研综述：多模态后训练与多模态 Agent 在 2024-2026 的发展脉络

### 1.1 多模态大模型架构：从"拼接"到"原生"

2024-2026 年间，多模态大模型（MLLM）架构经历了从"视觉编码器 + 投影器 + LLM"的拼接范式到逐步追求原生多模态理解的演进。2023 年的 BLIP-2 提出 Q-Former 作为视觉-语言连接器，LLaVA 则证明一个简单的 MLP 投影层就足够有效——这一发现深刻影响了后续架构设计。LLaVA-1.5（CVPR 2024 Highlight）进一步以仅 1.2M 公开数据 + 单日训练就达到 SOTA，确立了"CLIP-ViT + MLP + LLM"作为主流基线。

Qwen-VL 系列走出了一条三阶段训练（预训练 → 多任务微调 → 指令微调）的路线，Qwen2-VL（2409.12191）引入动态分辨率（NaViT-style）和 M-RoPE 位置编码，使模型能处理任意分辨率的图像。InternVL 系列则聚焦视觉编码器的规模化——从 InternVL 的 6B ViT 到 InternVL 1.5 的动态高分辨率，再到 InternVL3（2504.10479）和 InternVL3.5（2508.21732）将训练与推理配方推向极致。

2024 年的两个关键工作重新审视了 MLLM 设计的基本原则：MM1（ICML 2024, 2403.09611）通过大规模消融实验证明"连接器设计的影响微乎其微，图像编码器和数据质量才是关键"；Cambrian-1（NeurIPS 2024 Oral, 2406.16860）以视觉为中心系统评估了 20+ 视觉编码器，提出空间视觉聚合器（SVA）。NVLM（2409.11402）则探索了原生多模态架构，在单一 Transformer 中同时处理图文理解和生成。

### 1.2 多模态后训练：从 RLHF 到推理增强的完整栈

多模态后训练是 2024-2026 年发展最快的领域，形成了从数据合成到偏好对齐再到推理增强的完整技术栈。

**SFT 数据合成**方面，LLaVA 开创了利用 GPT-4 从图像标注生成指令数据的范式。ShareGPT4V（2311.12793）使用 GPT-4V 生成 100K 高质量描述并蒸馏到 1.2M 样本，证明更好的 Caption 直接提升下游性能。

**偏好对齐**方面，LLaVA-RLHF（2309.14525）首次将 RLHF 引入多模态，RLHF-V（CVPR 2024, 2312.00849）提出细粒度片段级人类修正配合密集 DPO，仅用 1.4K 数据就将幻觉降低 34.8%。Silkie/VLFeedback（EMNLP 2024, 2410.09421）首次构建大规模 AI 反馈数据集（82K 指令 + 理由），完全无需人类标注即提升 6.9%/9.5% 的感知/认知能力。MM-RLHF（2502.10391）进一步引入基于 Critique 的奖励模型和 120K 细粒度人类偏好对。一个值得关注的辩论是 Continual SFT（2411.14797）挑战了 RLHF 在多模态场景的必要性——仅靠负监督 SFT 就能匹配 RLHF 效果。

**推理增强**方面，受 OpenAI o1 启发，2024 年底出现了多模态"慢思考"范式。LLaVA-CoT（2411.10440）提出四阶段结构化推理（摘要 → 视觉解释 → 逻辑推理 → 结论），配合 test-time scaling search。Mulberry（2412.18319）引入集体 MCTS，多个模型协作搜索推理路径。Insight-V（2411.14432）用多 Agent + 迭代 DPO 生成无需人工标注的长推理链。

**幻觉**方面，评测从 POPE（首个系统性对象幻觉探测）发展到 HallusionBench（语言幻觉与视觉错觉纠缠诊断），缓解方法分为两条路径：解码时干预（VCD 对比解码、OPERA 过度信任惩罚——训练无关、即插即用）和训练时对齐（HII-DPO 用反事实图像、DA-DPO 难度感知优化）。2024 年的幻觉综述（2404.18930）对该领域做了系统梳理。

### 1.3 多模态 Agent：工具调用、Memory 与 RAG 的交叉地带

在多模态 Agent 领域（仅聚焦工具调用、Memory、RAG），2024-2026 出现了一些与多模态后训练可交叉的 work：

**工具调用**方面，MLLM-Tool（2401.10727）为多模态工具选择建立了基线，AvaTaR（NeurIPS 2024, 2406.11200）用对比推理优化工具使用，HAMMR（2404.05465）用层级化 ReAct 实现多 Agent 工具组合。

**Memory**方面，HippoRAG（NeurIPS 2024, 2405.14831）受海马体索引理论启发，用 LLM + 知识图谱 + Personalized PageRank 实现长期记忆，比 SOTA RAG 提升 20% 且成本低 10-30 倍。2026 年的 MemEye 和 MemLens 分别从评测角度揭示了当前记忆架构在视觉细节保持上的局限。

**多模态 RAG**方面，ColPali（ICLR 2025, 2407.01449）直接用 VLM 从文档页面图像生成多向量嵌入做迟交互检索，彻底绕过 OCR 流程。MRAG-Bench（ICLR 2025, 2410.08182）通过 16,130 图像 + 1,353 问题的评测揭示了一个重要发现：当视觉知识优于文本知识时，GPT-4o 仅获得 5.82% 提升而人类获得 33.16%——视觉 RAG 仍有巨大空间。

### 1.4 与组内研究方向的交叉点

作为主攻**多模态后训练**的学生，Agent 方向中最值得关注的是：
- **偏好对齐方法在 Agent 工具调用中的应用**（如 AvaTaR 的对比推理框架可借鉴到偏好数据构建）
- **RAG 中的检索-反思机制**（mR^2AG 的反思循环对多模态对齐的迭代改进有直接启发）
- **HippoRAG 的记忆架构**（对多模态对话系统中的偏好一致性有参考价值）
- **过程监督 RL 用于多轮工具调用**（对多模态 RLHF 的 credit assignment 问题有借鉴）

### 3.5 选材说明

**为什么这样分配？**

- **M2（后训练）占 22 篇**：这是你的主攻方向，我给了最大的篇幅。覆盖了 SFT 数据合成 → 偏好对齐（RLHF/DPO 完整路线）→ 推理增强（o1 范式）→ 幻觉（评测+缓解）四条子线。精读率 73%，确保对核心方法的深度理解。
- **M1（基础）占 13 篇**：目标不是穷举所有架构，而是建立从 BLIP-2 到 Qwen2-VL/NVLM 的全景认知。精读率 62%，关键架构精读、扩展架构粗读。

**总量的考量**：45 篇在 30-50 篇区间内，考虑到 M2 确实需要更多论文才能讲清楚完整的后训练演进脉络，22 篇是合理的选择。

## 八、调研方法自述

5. **WebFetch 直接验证**：对关键论文通过 arXiv HTML 页面直接确认标题、作者、贡献
