## 1. 新增模块 0：RL / RLHF / DPO / GRPO 基础补漏

**位置**：模块 1 之后、模块 2 之前。

**目录**：在当前目录下新建 `00_RL基础/`，包含 `papers/`、`notes/`、`README.md`。

**阅读顺序严格遵循**：PPO 教程 → InstructGPT → DPO → DeepSeek-R1（GRPO）

**真实的演进关系是**：PPO 是根，DPO 和 GRPO 都是从 PPO 长出来的分支（DPO 绕开 PPO，GRPO 简化 PPO）。所以必须先理解 PPO，再读 DPO 和 GRPO。

### 1.1 PPO 基础教程（非论文，1 天）

不下载论文。在 `00_RL基础/README.md` 中列出以下推荐资源链接（任选 1-2 份学习）：

- Costa Huang "The 37 Implementation Details of PPO"（强烈推荐，把 PPO 讲透）
- Lilian Weng "Policy Gradient Algorithms" 博客
- Hugging Face TRL 文档的 RLHF 章节

**目标**：看懂 PPO 目标函数、importance sampling ratio、clipping、KL penalty、critic 的作用。

### 1.2 InstructGPT 精读（arXiv:2203.02155，1.5 天，12h）

RLHF + PPO 的完整应用，是 LLaVA-RLHF 的语言模态前身。重点理解 reward model 训练、policy + reference + critic + KL penalty 的完整流程。

### 1.3 DPO 精读（arXiv:2305.18290，1.5 天，12h）

重点跟一遍从 RLHF 优化目标推导 DPO loss 的过程（Bradley-Terry 模型 + KL 约束闭式最优解）。这是 mDPO、RLHF-V、RLAIF-V 等论文的数学基础。

### 1.4 DeepSeek-R1 粗读（arXiv:2501.12948，1 天，3h）

重点理解 GRPO 与 PPO 的差异（去 critic、group-relative advantage 估计）。这是 Vision-R1、Visual-RFT 的算法基础。

### 1.5 下载与笔记

- 下载 InstructGPT、DPO、DeepSeek-R1 三篇论文到 `00_RL基础/papers/`，命名规范同其他模块
